Stuff I've Worked On

As a newbie to the industry, I have worked on many an independent projects! I've made both games and
applications. Click the download links below the description to try each app for yourself!

Slender
Awakening


A reboot and tribute to the classic indie horror
game, Slender: The 8 Pages. This game was
made using Unity Engine in just over a week, and
was completely coded and designed by me!

*I do not own any of the assets used in game,
they are under creative commons licensing.

Checkers! The
Game

This program was written in C# using a Windows Form. I created my
own rendition of the classic checkers with completely original art.
This game is online compatible, and can be played using an online
server application that the clients can connect to using TCP/IP. The
game took about a week to program, and was a fun way to learn
network programming along with game programming using C#!

"uwu" Android Translator

This app was a silly pet project of mine that I worked on mostly for fun. Me and friends enjoyed the cutesy "uwu"
language of the internet, so I programmed a translator for this "language" in Java! The application was tested on a
virtual machine and on my personal Android device, and was even submitted to the Google Play Store for a time!

TCP/IP Messenger

Yet another Windows Form C# Application, I love the .NET Framework! It makes it easy to create visually pleasing
looking applications that run smooth. This app allows users to connect to a server using a unique screen name and
message with their friends! Users can change the color of the text and change their username through an in-app
menu. The server application is simple and communicates each message along with the name of each user that
connects. Anyone can host a server as long as they forward a port and provide users with IP Address. This application
was challenging to develop but was a valuable learning experience!
